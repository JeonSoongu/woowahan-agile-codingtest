# ⚾ 미션 - 숫자 야구 게임

## 🎯 구현해야할 기능 목록
1. [v] 게임 초기화
2. [v] 중복되지 않는 컴퓨터 랜덤 숫자 셋팅(1 ~ 9 까지 서로 다른 임의의 수 3개)
3. [v]사용자 Input 값 가져오기
4. []사용자 입력받은 수의 유효성 검사
5. []스트라이크, 볼, 낫싱 카운트 
6. []결과물 출력
7. []정답 시 게임 종료하고 재시작 여부
8. []재시작 버튼 누르면 게임 재시작
